# SimDeck VS Code Extension

This extension opens the local SimDeck simulator inside a VS Code webview panel.

## Commands

- `SimDeck: Open Simulator View`
- `SimDeck: Stop Managed Server`
- `SimDeck: Show Output`

## Settings

- `simdeck.serverUrl`
- `simdeck.cliPath`
- `simdeck.port`
- `simdeck.bindAddress`
- `simdeck.autoStartServer`
